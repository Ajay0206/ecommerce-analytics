## PROJECT TITLE (for resume header)
E-Commerce Sales & Customer Analytics Dashboard
Tools: MySQL · Python (Pandas, NumPy, Matplotlib, Seaborn) · Power BI · Excel


## BULLET POINTS (pick 5–7 for a one-page resume)

• Designed and implemented a normalised MySQL relational database (6 tables,
  11,700+ records) with primary keys, foreign keys, and composite indexes to
  support high-performance analytical queries.

• Wrote 15+ SQL queries using JOINs, CTEs, aggregate functions, and window
  functions (LAG, RANK) to compute KPIs including Total Revenue (₹3.42 Cr),
  Average Order Value, Monthly Sales Trends, and Year-on-Year Growth.

• Built an end-to-end Python data pipeline using Pandas and NumPy for data
  cleaning (null handling, deduplication, outlier removal), feature engineering
  (customer tenure, order seasonality), and RFM segmentation of 500 customers
  into 6 behavioural segments.

• Performed Exploratory Data Analysis (EDA) and created 12 publication-quality
  visualisations using Matplotlib and Seaborn, covering revenue trends, category
  performance, regional analysis, customer demographics, and price distributions.

• Developed a 4-page interactive Power BI dashboard (Executive, Sales, Customer,
  Product) with 10+ DAX measures, cross-filter slicers, and KPI cards, enabling
  business stakeholders to self-serve insights without SQL knowledge.

• Conducted Customer Retention and Churn Analysis, identifying an 89% retention
  rate and 46.5% churn rate, and segmented customers using RFM scoring to
  highlight 124 Champion customers for targeted marketing campaigns.

• Implemented Region-wise and Category-wise Sales Analysis, surfacing that South
  India contributes the highest revenue share and Electronics is the top-grossing
  category, informing inventory and marketing strategy decisions.

• Automated dataset generation using Python's Faker library with reproducible
  random seeds, producing realistic Indian e-commerce data (customers, products,
  orders, shipping) suitable for repeatable analytics demonstrations.

Built an end-to-end e-commerce analytics project using MySQL, Python (Pandas,

KPIs, RFM customer segmentation, and a 4-page interactive BI dashboard.
